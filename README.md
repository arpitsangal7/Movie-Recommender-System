# 🎬 Movie Recommender System

## 📌 Overview
A personalized movie recommendation system using content-based and collaborative filtering techniques.

## 🚀 Features
- Suggests movies based on user preferences
- Uses similarity metrics for accurate recommendations
- Interactive UI using Streamlit

## 🛠 Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Streamlit

- Add hybrid recommendation system
- Improve UI
